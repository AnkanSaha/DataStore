## Run Locally

Clone the project

```bash
  git clone https://github.com/AnkanSaha/SaveNet.git
```

Go to the project directory

```bash
  cd DataStore
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run devstart
```


## Tech Stack

**Client:** HTML5, CSS3, Javascript, Pug

**Server:** Node, Express, MongoDB
